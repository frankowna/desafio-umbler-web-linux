# Pergunta 2 - Nginx, LiteSpeed e WordPress

## Objetivo

Esta seção apresenta uma proposta de configuração para hospedagem de aplicações WordPress utilizando Nginx como servidor web.

Além da configuração do Virtual Host, são apresentadas recomendações de segurança, desempenho e boas práticas normalmente utilizadas em ambientes de produção.

## Arquivos

- respostas.md → Explicação da arquitetura e das decisões adotadas.
- wordpress.conf → Exemplo de Virtual Host otimizado para WordPress.

## Premissas

A configuração prioriza:

- Segurança
- Performance
- Compatibilidade com WordPress
- Facilidade de manutenção
- Boas práticas de infraestrutura
